# nickanderson.us

Personal site, authored in R Markdown and rendered to `docs/` (served by GitHub
Pages; see `_site.yml` `output_dir: docs`). Rebuild with `rmarkdown::render_site()`
in R.

## Redirect tombstone (do not delete)

`economichistory.html` is a **static redirect stub** (no `.Rmd` source) that
points the long-dead `/economichistory.html` path at its successor
`research.html`. economichistory was a stale duplicate of research and was
removed from the nav; the stub keeps old inbound links from 404-ing.

It is **not** generated by `render_site()`, and `render_site()` may clean strays
from `docs/`. The canonical copy lives at the repo root (`economichistory.html`);
after any re-render, re-copy it into `docs/economichistory.html` so the redirect
survives:

```r
file.copy("economichistory.html", "docs/economichistory.html", overwrite = TRUE)
```
